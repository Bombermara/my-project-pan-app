My Project Pan - Privacy Policy

Ultima modifica: Giugno 2025

🛡️ La tua privacy è importante

My Project Pan è progettata per garantire il massimo rispetto della tua privacy. Tutti i dati inseriti nell’app rimangono esclusivamente salvati sul tuo dispositivo personale (iPhone o iPad).

📱 Dati raccolti

My Project Pan non raccoglie né trasmette alcun dato personale a server esterni o a terze parti.

  • ❌ Nessuna raccolta di dati sensibili
  • ❌ Nessun tracciamento delle attività
  • ❌ Nessuna condivisione con terze parti
  • ✅ Tutte le informazioni restano solo sul tuo dispositivo

💾 Archiviazione locale

L’app utilizza il sistema Core Data di Apple per salvare localmente:

  • Prodotti skincare inseriti manualmente
  • Routine personalizzate
  • Eventuali note, valutazioni o foto aggiunte dall’utente

I dati non vengono sincronizzati né inviati a server esterni.

🔒 Sicurezza dei dati

I dati sono protetti tramite le funzionalità di sicurezza integrate in iOS, inclusi:

  • Crittografia del dispositivo
  • Restrizioni di accesso a livello di sistema operativo
